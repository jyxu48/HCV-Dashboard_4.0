How to Use

  • Select a metric from the dropdown in the top bar — the map redraws to show that indicator across all PHAs.
  • Click any PHA dot on the map to load detailed performance data and a multi-year trend chart in the sidebar.
  • Hover over dots for a quick tooltip showing the agency name and selected metric value.
  • Use the Export CSV button in the sidebar to download the full PHA dataset.
